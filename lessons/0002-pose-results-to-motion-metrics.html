2. Joint angle

For shoulder A, elbow B, and wrist C:

u = A - B
v = C - B
theta = acos(dot(u, v) / (norm(u) * norm(v)))

This is a 2D projected angle. It is not automatically a true 3D anatomical angle.

3. Motion speed

velocity = (current_position - previous_position) / delta_time
speed = sqrt(velocity.x² + velocity.y²)

Without calibration or depth, the unit is pixels per second. Body-scale normalization can make comparisons more stable within a fixed camera setup.

5. What still needs implementation

  • Primary-person tracking and stable identity.
  • Confidence gating and trajectory smoothing.
  • Pure mathematical `IPoseAnalyzer` and `SquatAnalyzer`.
  • Camera calibration or depth for metric motion and reliable 3D geometry.

OpenCV provides a Kalman filter for future trajectory filtering and a camera calibration workflow for relating image points to a camera model.
